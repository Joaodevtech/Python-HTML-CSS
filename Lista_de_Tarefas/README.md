<h1>📝 Lista de Tarefas (To-Do List)</h1>

<p>
Projeto iniciante utilizando <strong>Python</strong>, <strong>Flask</strong>, <strong>HTML</strong> e <strong>CSS</strong>.<br>
Esta aplicação web permite <strong>adicionar</strong> e <strong>remover tarefas</strong> de forma simples, funcionando como um gerenciador de tarefas local.
</p>

<hr>

<h2>🌟 Funcionalidades</h2>

<ul>
  <li>✅ Adicionar novas tarefas</li>
  <li>❌ Remover tarefas existentes</li>
  <li>💻 Interface web simples e intuitiva</li>
  <li>⚙️ Executado localmente com Flask</li>
</ul>

<hr>

<h2>💻 Tecnologias Utilizadas</h2>

<ul>
  <li>🐍 Python 3</li>
  <li>🌐 Flask</li>
  <li>🧾 HTML5</li>
  <li>🎨 CSS3</li>
</ul>
